CADworks BIMXchange v4.0

BIMXchange affords a view into a powerful and intuitive Web based content app. It runs directly in Revit, and lets a user search, quickly find and insert into a BIM the content they need.

Getting Started

CADworks BIMXchange v4.0

The full featured CADworks BIMXchange is an intelligent, lightweight and intuitive Web application which provides easy access to over thirty-five thousand Revit® Family Types, from more than one hundred fifty building product manufacturers, and is currently utilized by several thousand Revit users. The Cloud based tool offers right-sized, rich-data Revit models of building products primarily important to the MEP designer, but with sufficient cross over products to be significant for architectural users as well. BIMXchange is available on a per seat basis, via a yearly subscription.

BIMXchange allows the Revit® user to access and navigate multiple libraries; either manufacturer specific, CADworks General, MEP discipline specific (mechanical, electrical & lighting, plumbing & fire protection), generic, or metric. Content selection begins with a tree view structured by categories, enhanced with renderings, then additionally by the ability to hierarchically search on all Shared Parameter values in each piece of content. Each Revit® Family Type has been built with engineering criteria for both project efficiency and collaboration, and to meet essential Revit® systems requirements. Once selected, content pieces are loaded swiftly, directly into a project from the CADworks Cloud, eliminating any need for a firm to manage or backup extensive content libraries. The Revit® user downloads only the objects they need.

To enable optimum design process and flow, CADworks will configure custom content libraries for any firm, or create new libraries, using a firm’s Revit® content, per the firm’s preferences. In addition, CADworks will brand BIMXchange to a firm’s image and penchant. All custom features are viewable only by a firm’s authenticated users, who use the same easily navigable drop-down selection menu for accessing distinct libraries.

Installation and Uninstallation

Please refer to our installation document.

To uninstall this plug-in, exit Revit if you are currenting running it, simply rerun the installer by downloading it again from Exchange, and select the 'Uninstall' button, or you can uninstall it from 'Control Panel\Programs\Programs and Features' (Windows 7) or Add/Remove Programs (Windows XP), just as you would uninstall any other application from your system.

Known Issues

  • Windows XP is not supported.
  • Some firewalls/proxy servers will block the BIMXchange, to correct this create an exception in the firewall/proxy to bxc.mepcontent.com.
  • Some Windows 7 users may have trouble installing the application if they are not the administrator of the workstation. If this is the case simply right-click on the installer once downloaded and select Run as Administrator.
